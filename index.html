Architecture Overview

One artifact, two deploy paths.

Diagrammed from the actual buildspec.yaml, appspec.yml, and Kubernetes manifests in the Java web-app project above.

Build stage. AWS CodeBuild installs Java 11 (Corretto) and runs the Maven build, producing a deployable .war artifact.
Path A — CodeDeploy. The artifact deploys to an EC2/Tomcat target via appspec lifecycle hooks (before-install, after-install, start, stop).
Path B — Kubernetes. The same app, containerized, runs as 3 replicas behind a NodePort Service — the cloud-native alternative to the EC2 path.
